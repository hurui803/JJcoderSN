

# 关于 go 安装说明:


## 开发环境搭建:

- [go 安装官方文档](https://golang.org/doc/install)
    - 必看, 此文档非常详细.


### Mac osx 开发环境配置:


-  go 安装:
    - 使用 homebrew 方式, 参看脚本.
- 配置 go 环境变量: (必要步骤)
    - go 安装路径
    - go 用户代码空间路径

- go 开发工具推荐:
    - IDEA ( jb 家的 IDE 全套都可以通过插件来支持 go, 但 idea 支持最佳)
    - sublime text3
